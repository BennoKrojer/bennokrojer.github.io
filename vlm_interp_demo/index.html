About This Viewer

This unified interface provides interactive access to four types of analyses across multiple model combinations:

  • LatentLens: Nearest contextual embeddings (words in real captions)
  • Logit Lens: Predicted tokens by applying LM head to intermediate layers
  • Embedding Matrix: Top-5 nearest vocabulary tokens for each visual patch
  • Patchscopes: Free-form descriptions generated by patching visual tokens
